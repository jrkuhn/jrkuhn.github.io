Projects


Flash Off

A Google Chrome Extension that freezes flashing or stimulating images in the hopes of preventing a seizure for those suffering from Photo-sensitive Epilepsy. The project was created in javascript using IM.js, a library which allows pixel-by-pixel image comparisons. The project also utilized the gif-frames.js library to extract individual images from gifs in the page for image comparison. The project was created within 72 hours during Iowa State Uniersity's HackISU Fall 2017.

Bubble Bunker

A dual-stick shooter created using Unity3D and C# in under 72 hours during Iowa State University's HackISU Spring 2017. The premise of the game is to shoot bubblegum at the attacking monsters and survive for as long as you can. I helped create the project in a team with two of my peers. I was responsible for much of the code as well as some of the 3D modeling and the Unity3D project layout.

Tabletop Calculator

A custom calculator webapp which targets mobile users looking for a better tabletop gaming tool than a general calculator. It contains two pop-up calculators with custom keypads, and several other features such as simulated dice rolling and coin flipping. I created the application as a personal project for me and my friends, and because it is a webapp, anyone with a mobile browser can quickly access it too.
